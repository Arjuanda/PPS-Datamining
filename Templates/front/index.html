Memperkenalkan PPS-Datamining

Penentuan program studi berdasarkan nilai siswa dengan algoritma data mining melibatkan penggunaan teknik-teknik data mining untuk menganalisis pola-pola dalam data nilai siswa dan memprediksi program studi yang paling cocok untuk setiap siswa berdasarkan karakteristik mereka.

Pada aplikasi ini akan ada 2 pengguna, yaitu admin dan calon mahasiswa. Aplikasi ini akan menyediakan menu login, dimana pengguna bisa mendaftar untuk pendataan sebagai calon mahasiswa yang sedang memilih program studi yang akan dijalani dengan memberikan data nilai yang menjadi acuan oleh sistem dalam merekomendasikan Program Studi kepada pengguna.
